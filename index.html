Projetos

Compra de Pizzas

Uma página web para compra de pizzas 100% responsiva. Utilizei as tecnologias: HTML | CSS | JavaScript. Foi realizado com JavaScript praticamente toda a estrutura, as imagens a quantidade de itens e elementos foram todos adicionados e posicionados com JavaScript. Também foi incluido nesse projeto um sistema de desconto que calcula na em tempo real sem precisar realizar qualquer ação.

Piano

Por fim, um dos meus projetos favoritos. Eu adoro tocar teclado, piano, então criar um piano com programação era uma meta a ser cumprida. Esse projeto foi feito pela primeira vez em HTML e CSS (link), lembro de estar aprendendo responsividade com flexbox ou grid, mas ainda não tinha conhecimento nehum com programação de fato. Depois de um tempo eu comecei a praticar e estudar JavaScript e desenvoli uma bateria, e depois que aprendi a lógica por trás de como manipular os elementos e colocar sons neles, ficou tudo mais fácil. Peguei esse projeto que tinha feito só com HTML e CSS e deixei ele funcional, divertido e 100% responsivo.
